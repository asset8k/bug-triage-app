# LLM Model Weights (Local-only)

This folder contains the fine-tuned LLM weights used by the GGUF-based engines.

These binary artifacts are typically excluded from Git commits (see the repository `.gitignore`).

## Required files (expected names)

- `llama_finetuned.gguf`
- `qwen_finetuned.gguf`

## Google Drive download links

The fine-tuned LLM weights are stored in Google Drive here:

- LLM models folder: https://drive.google.com/drive/folders/1qpyJjpyisDQzqzYO4OongFN20IugFFhC?usp=sharing

## After downloading

Place the downloaded files directly into this folder (`models/llm/`) so the filenames match exactly.

Quick check:
```bash
ls -la
```
